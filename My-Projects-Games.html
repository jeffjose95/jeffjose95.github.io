Space Puzzler


This is a game that I developed in 2 months using the Unity game Engine.

Initially, it was a project from one of my classes that I wanted to take to the next level.

There are 2 different versions of the game. One is meant for android and the other is for the web.

The goal of the game is to be able to move the ball around a platform and get it to the final spot. You get points for the total number of platforms that you passed over before getting to the end of the level

This game won the best theme based game award from the SJSU game development club in Spring 2017
